PHASE 6 STAGE 6.2 VALIDATION
============================

Stage:
- Stage 6.2: Autobiographical Synthesis

Mission:
- Make autobiographical memory reflect meaningful developmental continuity
  rather than isolated identity-bearing turns.


WHAT CHANGED
------------

1. Autobiographical reflection synthesis is now more explicit about continuity
   shape.
- The reflection engine can now classify autobiographical candidates as:
  - developmental milestones
  - continuity shifts
  - unresolved tensions
  - continuity notes

2. Generic explanatory assistant prose is rejected as autobiographical source
   material.
- Reflection synthesis now requires self-interpreting content and excludes
  generic explanatory patterns.

3. Direct autobiographical promotion from validated answers is more
   conservative.
- Raw assistant answers now need to look like stable self-claims before they
  are written directly into autobiographical memory.
- Analytical identity/tension answers are no longer written directly and are
  left for the reflection layer instead.

4. Semantic contamination was reduced.
- `relationship-context` semantic summaries are no longer used as
  autobiographical synthesis support.
- Autobiographical summary text now prefers episodic evidence preview instead
  of duplicating semantic summary text.
