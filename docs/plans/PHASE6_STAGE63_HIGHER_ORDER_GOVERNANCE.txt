PHASE 6 STAGE 6.3 VALIDATION
============================

Stage:
- Stage 6.3: Higher-Order Memory Governance

Mission:
- Make higher-order continuity memory more trustworthy by improving provenance,
  confidence, conflict resolution, and retention controls.


WHAT CHANGED
------------

1. Higher-order semantic and autobiographical records now carry explicit
   governance metadata.
- Reflection-produced records now preserve:
  - governance scope
  - claim axis
  - claim value
  - support count
  - source channels
  - source event ids
  - evidence preview

2. Higher-order merge behavior now distinguishes reinforcement from
   contradiction.
- Repeated support for the same higher-order claim merges into the active
  record with accumulated provenance.
- Conflicting active higher-order claims are no longer silently overwritten.
- Older conflicting semantic and autobiographical records are archived with:
  - `governance_status: superseded`
  - `superseded_by`
  - `superseded_at`
- Newer active records retain explicit `supersedes_conflicts` references.

3. Retrieval now prefers newer, better-supported governed higher-order records.
- Semantic and autobiographical retrieval now incorporate:
  - support count
  - governance status
  - continuity weight
  - confidence
- Archived higher-order records remain inspectable but are down-ranked.


TEST COVERAGE
-------------

Added/updated tests verify:
- semantic candidates carry governance scope and provenance support count
- autobiographical candidates carry governance scope
- conflicting semantic higher-order updates archive the prior active record
- conflicting autobiographical higher-order updates archive the prior active
  record
- retrieval prefers the newer active autobiographical conflict winner over the
  archived predecessor
